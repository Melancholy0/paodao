# 鼠鼠日记

一个轻量的《三角洲行动》跑刀记录小站，支持历史记录、计时、出货统计与自动补全。

## 功能
- 跑刀历史记录与搜索
- 跑刀计时（开始/结束）
- 出货记录与合并统计
- 进度百分比、剩余目标显示
- 本地保存 + 导入/导出 JSON

## 使用
直接打开 `index.html` 即可。

可选脚本：
- `scrape.js` / `scrape_keys.js`：爬取红色收藏品、房卡名称
- `update_suggestions.js`：把爬取结果写回 `app.js`

## 结构
- `index.html` 页面
- `styles.css` 样式
- `app.js` 主逻辑
- `red_items.json` / `key_items.json` 数据
本项目提供一个本地网页，用于记录三角洲行动跑刀的哈夫币目标与摸金明细。

## 使用方式

1. 直接用浏览器打开 `index.html`。
2. 填写“本次准备跑多少哈夫币”“目前跑了多少”等信息。
3. 在“摸金明细”中选择分类、填写名称与价值并添加，即可自动汇总。

数据仅保存在浏览器内存中，关闭页面后会清空。若需要持久化，可后续扩展为本地存储。
