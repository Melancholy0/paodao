# 鼠鼠日记

一个轻量、可视化的《三角洲行动》跑刀记录小站：记录每次跑刀、统计进度、保留出货明细与计时数据。

## 功能
- 历史记录：搜索、打开、删除
- 跑刀明细：目标/余额/进度/剩余/百分比
- 出货记录：红色收集品/房卡/其他，自动补全与合并统计
- 计时与结单：自动保存耗时、备注与进度
- 本地存储：浏览器本地保存，支持 JSON 导入/导出

## 快速使用
1. 打开 `index.html`
2. 点击“开始跑刀”，填写 游戏ID 与 UID
3. 进入明细后开始计时并记录出货
4. 剩余为 0 后点击“结单”保存

## 单文件版
- 直接打开 `single_html/index.html` 即可使用（无需其他文件）
- 数据仍保存在浏览器本地，可继续使用导入/导出功能

## 数据与脚本
- `red_items.json` / `key_items.json`：红色收集品与房卡名称
- `scrape.js` / `scrape_keys.js`：抓取名称
- `update_suggestions.js`：写回到 `app.js`
